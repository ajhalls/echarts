ECharts features introduced

ECharts, a pure Javascript charting library can run smoothly on a PC and mobile devices, compatible with most of the current browsers (IE8 / 9/10/11, Chrome, Firefox, Safari, etc.), dependent on the underlying light middleweight Canvas library ZRender , intuitive, vivid, interactive, highly personalized custom data visualizations.

ECharts 3 more are added to a richer and more interactive features visual effects, and optimized for mobile end depth.

Chart types galore

ECharts provide a general line chart , histogram , scatterplot , pie , K line graph , for statistics boxplot , for geographic data visualization map , thermodynamic diagram , chart , diagram , treemap , multi-dimensional data visualization parallel coordinates , as well as for the BI funnel plot , dashboard , and supports map and map of the mashup between.

When you can download all the charts contained in the download file to build the interface, if you just need one or two diagrams, they complain that contains all the file is too large to build the chart, you can also choose to build online requires a custom chart types Construct.

support multiple coordinate systems

ECharts 3 became independent of the concept of "coordinate" the support of the Cartesian coordinate system (catesian, with grid), polar coordinates (polar), geographic coordinate system (geo). Charts can be used across a coordinate system, for example, fold, column, scatter diagram, etc. can be placed on a Cartesian coordinate system can be placed on a polar coordinate system, or can be placed in the geographic coordinate system.
